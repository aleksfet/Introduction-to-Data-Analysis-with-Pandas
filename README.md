# Module 9 Assignment: Data Analysis with Pandas

## 📊 Overview
This project analyzes sales data for a fictional company called **GlobalTech**.

The goal is to use **Pandas** to explore, filter, and analyze data to understand sales performance across regions, products, and categories.

---

## 🧠 What This Project Does

- Loads and explores sales data from a dataset
- Calculates total revenue, units sold, and average prices
- Filters data based on conditions (region, product, promotions)
- Identifies top-performing products and regions
- Analyzes category performance and promotion effectiveness
- Detects missing values and data quality issues
- Generates a business report with insights and recommendations
